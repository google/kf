## kf set-env

Set an environment variable for an app

### Synopsis

Set an environment variable for an app

```
kf set-env APP_NAME ENV_VAR_NAME ENV_VAR_VALUE [flags]
```

### Examples

```
  kf set-env myapp ENV production
```
