## kf create-service-broker

Add a cluster service broker to service catalog

### Synopsis

Add a cluster service broker to service catalog

```
kf create-service-broker BROKER_NAME URL [flags]
```

### Examples

```
  kf create-service-broker mybroker http://mybroker.broker.svc.cluster.local
```
